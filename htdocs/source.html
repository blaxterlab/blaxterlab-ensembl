Source code

The Lepbase ensembl instance is built on the Ensembl and EnsemblGenomes codebase with additional functionality from SequenceServer (BLAST server) and Web Apollo (community annotation)
Our Ensembl-specific modifications to the webcode and our data import pipeline (which includes search indexing) are hosted in the lepbase-ensembl and lepbase-import repositories on BitBucket
We have written a generic GFF parser, gff_parser, which we use in our import pipeline to accommodate, validate and correct the diversity of GFF formats used in Lepidopteran genome projects
We have customised SequenceServer to allow closer integration with our Ensembl instance in the lepbase-sequenceserver repository
Our assembly statistic visualisations are based on our assembly_stats javascript project
We have written an API training environment, lepbase-workshop, for use in training workshops, which we plan to make more widely applicable as an introduction to the Lepbase API
