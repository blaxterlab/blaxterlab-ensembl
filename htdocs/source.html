Source code

At Lepbase we support open source and open data. We endeavour to make all the data we host available for bulk download and our tools are powered by and released as open source software.
The source code for all our projects is available on or
Our modifications to ensembl-webcode for this site are available at lepbase-ensembl
Our data import pipeline (which includes search indexing) is available at lepbase-import
We have written a generic GFF parser, gff_parser, which we use in our import pipeline to accommodate, validate and correct the diversity of GFF formats used in Lepidopteran genome projects
Our fork of SequenceServer with changes to allow closer integration with our Ensembl instance is available at lepbase-sequenceserver
Our assembly statistic visualisations are based on our assembly_stats javascript project
We have written an API training environment, lepbase-workshop, for use in training workshops, which we plan to make more widely applicable as an introduction to the Lepbase API
Lepbase uses many other open source bioinformatics projects, including:
The EBI's Ensembl/EnsemblMetazoa genome browser powers ensembl.lepbase.org

Ensembl 2015. Cunningham F, et al. 2015. Nucleic Acids Research 43:D662-D669
blast.lepbase.org is powered by the Wurm Lab's SequenceServer

SequenceServer: BLAST search made easy, Priyam A, Woodcroft BJ, Rai V, Wurm Y. in prep. http://sequenceserver.com
Community annotations at webapollo.lepbase.org are made possible by Web Apollo

Web Apollo: a web-based genomic annotation editing platform. Lee E, Helt GA, Reese JT, Munoz-Torres MC, Childers CP, Buels RM, Stein L, Holmes IH, Elsik CG, Lewis SE. 2013. Genome Biol 14:R93
