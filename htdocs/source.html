Source code

The Lepbase ensembl instance is built on the Ensembl and EnsemblGenomes codebase, available on GitHub
Our Ensembl-specific modifications to the webcode and our data import pipeline (which includes search indexing) are hosted in the lepbase-ensembl and lepbase-import repositories on BitBucket
We have written a generic GFF parser, gff_parser, which we use in our import pipeline to accommodate, validate and correct the diversity of GFF formats used in Lepidopteran genome projects
Our BLAST server is powered by SequenceServer with modifications to allow closer integration with our Ensembl instance in the lepbase-sequenceserver repository
We use Web Apollo to allow community annotation of each genome that we host
Our assembly statistic visualisations are based on our assembly_stats javascript project
As an Ensembl site, we have an API which is available but any lepbase-specific details are currently undocumented. We have written an API training environment, lepbase-workshop, for use in training workshops, which we intend to make more widely available as an introduction to the Lepbase API
