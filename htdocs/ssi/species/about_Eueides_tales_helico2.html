About Eueides tales

Part of the 18 Heliconiine DISCOVAR assembly project. These are preliminary data and have not been published or verified. Please use blast.lepbase.org to search for your region of interest, and webapollo.lepbase.org to contribute any manual annotations. Please contact James Mallet before using any data from this assembly in a publication.

Assembly

This genome assembly was created using DISCOVAR by David Jaffe. DISCOVAR uses long (250 bp in this case) overlapping paired-end reads to create highly accurate contigs. These assemblies have not been checked for contaminants, redundancy, repeats, or heterozygosity. All the DISCOVAR heliconiine assemblies have >100,000 contigs, many of which are smaller than 250bp and are possibly redundant.

Annotation

For this release of Lepbase, no gene predictions have been done as the DISCOVAR assemblies are highly fragmented. Please contact us at help@lepbase.org if you are interested in a full genome annotation. If there is sufficient interest, we will add annotations to future Lepbase releases.
